# 变更日志

所有重要的变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.4.0] - 2025-01-30

### 新增
- 多关键字 AND 搜索：空格分隔的关键字同时匹配
- 引号精确匹配：`"项目会议"` 作为整体匹配
- 右键删除功能：支持删除书签、关闭标签页、删除历史记录、删除下载记录
- 删除确认弹窗显示具体名称：删除前显示书签名、标签名等便于确认
- 打包工具脚本：`scripts/build.sh` 一键生成 Chrome Web Store 上传包

### 优化
- 右键菜单定位优化：底部右键时菜单自动向上弹出，不再被遮挡

### 修复
- 修复右键菜单在底部项目上被裁剪的问题
