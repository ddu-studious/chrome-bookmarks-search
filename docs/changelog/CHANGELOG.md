## [1.6.0] - 2025-02-03

### 新增
- **全局浮层搜索**：Content Script 注入模式，在任意网页覆盖式搜索
- **三种 UI 风格**：Spotlight (macOS)、Raycast (命令面板)、Fluent (Windows 11)
- **全新设置页面**：侧边栏导航的完整设置中心 (options.html)
- **数据管理功能**：在设置中查看、搜索、筛选、管理书签/历史/下载
- **使用说明手册**：内置文档，包含快捷键指南、搜索语法、界面风格说明
- **友情链接管理**：支持添加、编辑、删除快捷链接
- **风格化右键菜单**：三种风格各有专属右键菜单样式，带快捷键提示

### 优化
- 搜索浮层尺寸调整为 620px 宽，更紧凑
- 右键菜单边界检测，自动调整位置避免遮挡
- 快捷键统一为 Alt+B (Mac: Option+B)

### 技术变更
- 从 popup 模式迁移到 content script 全局浮层模式
- 新增 options.html 和 js/options.js
- 新增 css/options.css
- manifest.json 新增 options_page、scripting 权限

## [1.5.0] - 2026-02-02

### 新增
- **书签健康分析**：识别从未使用、很少使用、休眠的书签
- **书签筛选器**：快速筛选「未使用」「很少使用」「休眠」书签
- **状态标签**：直观显示书签使用状态（从未访问、访问较少、长期未访问）
- **添加时间显示**：未使用书签显示添加时间，帮助判断是否过时

### 技术细节
- 书签分类标准：
  - 未使用：访问次数 = 0
  - 很少使用：访问次数 ≤ 2
  - 休眠：超过 180 天未访问
  - 活跃：其他情况
- 筛选器仅在书签模式下显示
- 复用现有访问统计数据，无额外性能开销

## [1.4.1] - 2025-01-31

### 新增
- 书签编辑功能：右键菜单支持编辑书签的标题和网址
- 编辑弹窗：优雅的内联编辑界面，支持 ESC 取消、Enter 保存
- 智能菜单：编辑选项仅在书签模式下显示

### 说明
- 标签页、历史记录、下载记录为只读数据，不支持编辑
- 编辑功能通过 Chrome Bookmarks API 实现
