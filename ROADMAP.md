# Chrome Bookmarks Search 优化路线图

## 项目愿景
打造一个快速、高效、智能的浏览器资源搜索工具，提供现代化的用户体验。

## 版本规划

### v1.0.0 [已发布]
- [x] 基础搜索功能
- [x] 键盘导航
- [x] 多类型资源支持（书签、标签页、历史、下载）
- [x] 基础 UI 交互
- [x] Favicon 支持

### v1.1.0 [已发布]
- [x] 深色模式支持
  - [x] 自动跟随系统主题
  - [x] 手动切换主题
  - [x] 主题切换动画
- [x] 字体调节功能
  - [x] 字体大小调节（小/中/大）
  - [x] 行间距优化
- [x] 基础辅助功能
  - [x] 键盘操作提示
  - [x] 焦点高亮优化
  - [x] 动画控制选项

### v1.2.0 - 搜索体验优化 [已完成]
- [x] 高级搜索语法
  - [x] site: 语法支持
  - [x] type: 语法支持
  - [x] in: 语法支持
  - [x] after:/before: 时间过滤
- [x] 智能排序
  - [x] 访问频率排序
  - [x] 时间排序
  - [x] 相关度排序
- [x] 搜索设置持久化
  - [x] 记住默认搜索模式
  - [x] 记住排序偏好

### v1.3.0 - 快捷操作增强 [已完成]
- [x] 批量操作支持
  - [x] 多选功能 (Shift/Ctrl + 点击)
  - [x] 批量打开
- [x] 右键菜单增强
  - [x] 在新标签页打开
  - [x] 在隐私窗口打开
  - [x] 复制链接
  - [x] 分享链接
- [x] 右键删除功能

### v1.4.0 - 多关键字搜索 [已完成]
- [x] 多关键字 AND 搜索
- [x] 精确短语匹配（引号语法）
- [x] 右键菜单定位优化

### v1.6.0 - 全局浮层搜索 [已完成]
- [x] 全局浮层搜索模式
  - [x] Content Script 注入
  - [x] 任意页面唤起搜索
- [x] 三种 UI 风格
  - [x] Spotlight (macOS 风格)
  - [x] Raycast (命令面板风格)
  - [x] Fluent (Windows 11 风格)
- [x] 全新设置页面
  - [x] 侧边栏导航
  - [x] 数据管理列表
  - [x] 使用说明手册
- [x] 风格化右键菜单

### v1.7.0 - 智能推荐 [规划中]
- [ ] 上下文感知
  - [ ] 基于当前标签推荐
  - [ ] 同域名页面推荐
- [ ] 快速访问区
  - [ ] 最常访问站点
  - [ ] 最近关闭标签页
- [ ] 智能分组
  - [ ] 工作/学习分组
  - [ ] 自定义分组

## 优先级说明
1. 🏃‍♂️ 最高优先级：v1.7.0 智能推荐
   - 增强用户体验
   - 智能化功能
   - 个性化推荐

2. 🚶‍♂️ 中等优先级：性能优化
   - 大量书签搜索性能
   - Favicon 缓存策略
   - 首次加载优化

## 注意事项
1. 保持界面简洁，避免功能过度堆积
2. 新功能需要提供适当的用户引导
3. 保持性能优化，避免功能影响加载速度
4. 定期收集用户反馈，调整优化方向
