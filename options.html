使用说明

快速上手 Chrome Bookmarks Search

快捷键指南

Alt + B打开搜索
↑ ↓上下选择
← →切换模式
Enter打开选中
Esc关闭窗口
⌘ Click多选

搜索语法

关键字1 关键字2 多关键字 AND 搜索
site:github.com 限定网站搜索
in:title / in:url 限定搜索范围
after:2024-01 指定日期之后
before:2024-06 指定日期之前
type:pdf 文件类型过滤

界面风格

点击搜索浮层右下角的调色盘按钮切换风格：

  • Spotlight - macOS 风格，毛玻璃效果
  • Raycast - 命令面板风格，深色主题
  • Fluent - Windows 11 风格，亚克力效果

书签状态

  • 未使用 - 从未访问过的书签
  • 很少使用 - 访问次数少于3次
  • 休眠 - 超过180天未访问
  • 活跃 - 正常使用的书签
